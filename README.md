# MPP Portal Screenshot Showcase

An elegant, single-page screenshot viewer for the MPP Portal with English/Spanish language toggle.

## Live Demo

**URL:** https://charlesmartinedd.github.io/mpp-portal-showcase/

## Features

- Dark mode design inspired by Apple/Microsoft aesthetics
- 18 portal screenshots organized in a 6x3 grid
- English/Spanish language toggle (persists in browser)
- Full-screen modal viewer with keyboard navigation
- Embedded Articulate training course
- No scrolling required - viewport-contained layout

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `ESC` | Close modal |
| `←` | Previous screenshot |
| `→` | Next screenshot |

## For Contractors

To update screenshots:
1. Replace images in `images/en/` (English) or `images/es/` (Spanish)
2. Keep the same filenames (e.g., `01-homepage.png`)
3. Commit and push changes

## Local Development

Simply open `index.html` in any modern browser. No build process required.
